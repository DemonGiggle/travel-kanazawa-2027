金澤 6天5夜（2027）

不自駕、順路版本：TPE → KIX → 金澤 → 京都 → KIX → TPE。下面為每日重點與景點說明。

Day 1｜TPE → KIX → 金澤

早上

從台北出發，飛往關西機場。建議確認行李與入境資料，預留通關時間。

中午

抵達 KIX，入境後搭車前往金澤（約2–3小時，視交通方式而定）。

下午

辦理入住並休息。若抵達時間允許，可在金澤站周邊散步並熟悉交通路線。

晚上

在金澤站或香林坊附近用晚餐，準備隔天行程。

交通：TPE → KIX → 關西機場搭車 → 金澤站 → 飯店
Day 2｜白山一里野溫泉滑雪場（雪日）

全天以滑雪場活動與溫泉體驗為主，建議提前預訂雪具與溫泉住宿。

白山一里野滑雪場

滑雪與雪上活動，適合全家。初學者可預約教練或租借容易上手的雪具。

一里野溫泉

晚間安排溫泉與當地料理，放鬆肌肉並體驗日式旅館服務。

交通：金澤飯店 → 白山一里野（巴士或包車）→ 溫泉住宿
Day 3｜兼六園 → 金澤城公園 → 21世紀美術館

兼六園（早晨）

日本三大名園之一，冬季常見雪景。建議步行路線並注意地面濕滑。

金澤城公園

可參觀城郭與庭園，拍攝建築與景觀照。

21世紀美術館（下午）

現代藝術展覽為室內行程的好選擇，事先線上查展覽資訊並預約門票（如需）。

交通：一里野 → 金澤市區（巴士）→ 步行或短程公車
Day 4｜近江町市場 → 金箔體驗 → 東茶屋街 → 前往京都

近江町市場（早餐/早午餐）

海鮮丼與當地海產為招牌，建議早到以避免人潮高峰。

金箔工藝／手作體驗

短時的金箔貼飾體驗，適合帶回作為紀念品；請事先預約。

東茶屋街

保存良好的歷史街區，適合散步、喝茶與購買手信。

傍晚交通：金澤 → 京都（電車）
Day 5｜京都：清水寺、祇園、錦市場

清水寺與二年坂/三年坂

歷史景點與古風商街，建議穿舒適鞋子並留意階梯與坡道。

祇園與花見小路

傍晚的散步路線，可能會遇到舞妓或藝伎行經的街景。

錦市場

傳統食材與小吃集中地，適合購買伴手禮與品嘗當地小吃。

交通：京都市區步行與公車/地鐵搭配
Day 6｜京都半日 → KIX → TPE

上午：京都站或京都鐵道博物館

視團體需求選擇：親子可前往鐵道博物館；行李收整需預留時間。

離程

回飯店取行李，前往關西機場搭機返回台北。

交通：京都飯店 → 京都站 → KIX → TPE

行程內容以景點與交通為主，已依天分段與景點說明，便於現場操作與時間安排。
