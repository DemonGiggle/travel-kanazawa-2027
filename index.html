金澤暖心 6天5夜（2027）

不自駕、順路版本：TPE → KIX → 金澤 → 京都 → KIX → TPE。每日內容可摺疊，畫面簡潔。

概要

行程總覽：6天5夜。交通順序：TPE → KIX → 金澤 → 京都 → KIX → TPE。偏溫馨風格，適合家庭與朋友。

Day 1｜TPE → KIX → 金澤

早上

從台北出發，飛往關西機場。

中午

抵達 KIX，入境後直接搭車前往金澤。

下午

以移動為主，抵達金澤後先辦理入住、稍微休息。若時間允許，可在金澤站周邊逛逛。

晚上

金澤站附近吃晚餐，早點休息。

交通順序：TPE → KIX → 關西機場搭車 → 金澤站 → 飯店
Day 2｜白山一里野溫泉滑雪場玩雪 → 住一晚

早上

從金澤出發前往白山一里野溫泉滑雪場。建議早起出門，這天主打玩雪、雪橇、拍照。

中午

在雪場周邊簡單吃午餐，繼續玩雪。

下午

玩到下午後，移動到一里野溫泉區住宿。入住後稍作休息。

晚上

泡湯、吃晚餐，這天就輕鬆收尾。

交通順序：金澤飯店 → 金澤站/巴士站 → 白山一里野溫泉滑雪場 → 一里野溫泉住宿
Day 3｜一里野 → 金澤市區：兼六園 → 金澤城公園 → 21世紀美術館

早上

從一里野回到金澤市區。先去兼六園，冬天雪景很適合慢慢走。

中午

步行順遊金澤城公園，附近找地方吃午餐。

下午

前往21世紀美術館，排室內行程剛剛好。

晚上

回金澤站或香林坊一帶吃晚餐。

交通順序：一里野住宿 → 巴士回金澤 → 兼六園 → 金澤城公園 → 21世紀美術館 → 市區飯店
Day 4｜近江町市場 → 金箔工藝／手作 → 東茶屋街 → 京都

早上

先去近江町市場吃早餐或早午餐，海鮮丼很適合排這天。

中午

前往金澤金箔工藝館或安排金箔手作體驗。

下午

到東茶屋街散步、喝茶、買小東西。傍晚從金澤移動到京都。

晚上

抵達京都後入住，附近簡單吃飯。

交通順序：飯店 → 近江町市場 → 金箔工藝館/手作體驗 → 東茶屋街 → 金澤站 → 京都 → 京都飯店
Day 5｜京都經典散步日

早上

前往清水寺，順走二年坂、三年坂。

中午

在清水寺/祇園周邊吃午餐。

下午

逛祇園、花見小路，之後可接錦市場或四條河原町。

晚上

京都市區晚餐，最後買伴手禮。

交通順序：京都飯店 → 清水寺 → 二年坂三年坂 → 祇園/花見小路 → 錦市場/四條河原町 → 京都飯店
Day 6｜京都半日 → KIX → TPE

早上

安排一個京都站附近、收尾方便的點。最適合親子的是京都鐵道博物館；若想輕鬆，可改成京都站周邊逛街。

中午

回飯店拿行李，前往關西機場。

下午/晚上

搭機回台北。

交通順序：京都飯店 → 京都鐵道博物館/京都站周邊 → 京都站 → KIX → TPE

此頁為簡易暖心版行程。祝旅途愉快！
